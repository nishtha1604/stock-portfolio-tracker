🔐 Stock Portfolio Tracker

📌 Overview

This project demonstrates a simple Stock Portfolio Tracker developed using Python. The application allows users to enter stock symbols and quantities, calculate the total investment value, and store the portfolio summary in a text file.

The project focuses on implementing fundamental Python concepts such as dictionaries, loops, user input handling, arithmetic operations, and file handling.
 
🎯 Objectives

- Develop a basic stock portfolio management system  
- Calculate total investment value using predefined stock prices  
- Understand the use of dictionaries in Python  
- Implement user input and output handling  
- Practice file handling by saving portfolio summaries  
- Improve problem-solving and logical thinking skills  


✨ Features

✅ Accepts user input for stock symbols and quantities  
✅ Uses a predefined dictionary for stock prices  
✅ Calculates investment value for each stock  
✅ Displays total portfolio investment  
✅ Saves portfolio summary in a `.txt` file  
✅ Simple and beginner-friendly console interface  

💻 Technologies Used

- Python  

🧠 Concepts Implemented

- Dictionaries  
- Loops  
- Conditional Statements  
- User Input/Output  
- Arithmetic Operations  
- File Handling
